#### Nous avons décidé de faire un serveur de jeux de cartes.

### Compte rendu n°1

Pour le moment nous avons pensé à la contrée, nous voulons donc tout automatiser, tout autant dans le comptage des points que dans la verification du bon enchainement des cartes selon les règles du jeu.

Nous pourrons ensuite partir sur la belote, ou encore le Uno, ou même, pourquoi pas, créer un jeu avec nos propres règles.

Nous avons décidé de choisir ce type de jeu, jeu de carte, car celui-ci demande des interactions en live sur tous les clients, ainsi grâce à React et Node.js ceci est possible.

Pour le moment nous préférons partir sur un système de jeux ouvert, c'est à dire qu'il n'y a pas besoin de compte pour jouer, uniquement un username, ceci pourra être modifié dans l'avenir.
